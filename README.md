# 올영세일 쇼핑몰형 재고 알림 데모 Plus

기존 `올영세일 사용자 앱 데모`를 실제 쇼핑몰처럼 보이도록 확장한 GitHub Pages용 정적 파일입니다.

## 추가된 핵심 기능

- 홈 화면: 세일 배너, 빠른 카테고리 이동, 시간대 쿠폰 미리보기, 인기 상품 랭킹
- 카테고리 화면: 전체/스킨케어/세럼/선케어/메이크업/마스크팩/헤어케어/바디케어/푸드/뷰티소품 탭 이동
- 상품 카드: 브랜드, 상품명, 할인율, 정가/판매가, 리뷰 수, 오늘드림/쿠폰/품절임박 배지
- 상품 상세 모달: 상품 혜택, 재고 상태, 장바구니 담기
- 쿠폰센터: 0시/12시/18시 선착순 쿠폰 발급 데모
- 장바구니: 수량 변경, 삭제, 재고 상태 표시, 쿠폰 선택/적용, 결제 예정 금액 계산
- 결제하기: 배송지 입력 → 결제수단 선택 → 주문 완료 3단계 시뮬레이션
- 알림함: 품절 위험, 품절, 재입고 알림 이력 표시
- 발표자 패널: LIVE 시작/중지, 쿠폰 전체 오픈, 재고 하락, 재입고, 초기화

## 배포 방법

1. 기존 GitHub Pages repository에서 `index.html`을 백업합니다.
2. 이 파일의 `olive_sale_plus_index.html` 이름을 `index.html`로 변경합니다.
3. 기존 repository에 업로드합니다.
4. GitHub Pages URL을 새로고침합니다.

## API 연결 방식

기본적으로 기존 API Gateway 주소를 그대로 사용합니다.

```js
const API_BASE_URL = "https://ukxqcqhsr9.execute-api.ap-northeast-2.amazonaws.com";
```

현재 API가 응답하지 않아도 화면은 로컬 더미 데이터로 계속 작동합니다. 그래서 발표 중 API가 잠깐 안 되어도 쇼핑몰 화면, 쿠폰, 장바구니, 결제 시뮬레이션은 유지됩니다.

## 발표 시연 추천 순서

1. 홈에서 세일 배너와 쿠폰 영역 보여주기
2. 카테고리 탭을 눌러 다양한 상품이 바뀌는 모습 보여주기
3. 상품 상세 열기 → 장바구니 담기
4. 쿠폰센터에서 0시/12시/18시 쿠폰 발급
5. 장바구니에서 쿠폰 적용 후 결제 금액 변화 확인
6. 결제하기에서 배송지 → 결제수단 → 주문완료까지 진행
7. 발표자 패널에서 재고 하락 클릭 → 품절 위험 알림 확인

## 주의

- 실제 결제는 발생하지 않는 데모입니다.
- 실제 올리브영 UI를 그대로 복제하지 않고, 발표용으로 유사한 쇼핑 경험만 재구성했습니다.
- 상품/쿠폰/재고 데이터는 데모용 임의 데이터입니다.
