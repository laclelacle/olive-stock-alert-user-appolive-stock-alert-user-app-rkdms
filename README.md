# 올영세일 사용자 앱 데모

## 구조
- 왼쪽: 실제 사용자가 보는 모바일 앱 화면
- 오른쪽: 발표자용 컨트롤 패널

## 주요 기능
- 홈 / 장바구니 / 알림 / 결제 탭
- 장바구니 상품 실시간 재고 표시
- 1초마다 재고 조회
- 3초마다 재고 하락 시뮬레이션
- 10개 미만 품절 위험 알림
- 0개 품절 처리
- 재입고 처리
- 발표용 결제 화면

## 배포
GitHub Pages용 정적 파일입니다.
새 repository를 만들고 index.html과 README.md를 업로드한 뒤 Pages를 켜면 됩니다.


## 추가 수정: 앱 전체 배너 알림
- 홈/장바구니/알림/결제 어느 탭에 있어도 새 품절 위험·품절·재입고 알림이 상단 배너로 표시됩니다.
- 배너를 누르면 알림 탭으로 이동합니다.
- 품절 위험은 주황 배너, 품절은 빨간 배너, 재입고는 파란 배너로 구분됩니다.


## 수정: iPhone 스타일 배너 알림
- 재고가 부족하지 않으면 배너가 뜨지 않습니다.
- 재고가 10개 미만으로 떨어져 새 알림이 생성될 때만 상단에 iPhone 알림처럼 짧게 표시됩니다.
- 품절/재입고 알림도 같은 배너 형태로 표시됩니다.
- 배너를 누르면 알림 탭으로 이동합니다.
