# 올영세일 모바일 앱형 데모

기존 재고 알림 데모를 올리브영 모바일 앱 느낌으로 재구성한 GitHub Pages용 정적 파일입니다.

## 포함 기능
- 모바일 앱형 상단 헤더, 검색창, 가로 탭, 하단 탭바
- 홈 / 오특 / 랭킹 / 세일 / 기획전 / AI픽 / 이벤트 탭 전환
- 앱 드로어 느낌의 전체 카테고리 사이드바
- 공식몰 확인 상품명·정가·판매가·이미지 URL 기반 상품 카드
- 상품상세 하단 시트
- 0시 / 12시 / 18시 타임쿠폰 발급
- 장바구니 수량 변경, 쿠폰 적용, 최종금액 계산
- 주문/결제 3단계 데모
- 마이페이지 주문/배송 조회 데모
- 사용자가 만든 올리브 캐릭터 내장

## 배포 방법
1. `index.html`과 `README.md`를 GitHub repository 최상단에 업로드합니다.
2. GitHub Pages 설정에서 `main / root`로 배포합니다.
3. 1~3분 뒤 Pages URL을 새로고침합니다.

## 주의
교육/발표용 데모이며 실제 결제는 발생하지 않습니다. 실제 상품 가격은 세일, 쿠폰, 재고, 프로모션에 따라 변동될 수 있습니다.
