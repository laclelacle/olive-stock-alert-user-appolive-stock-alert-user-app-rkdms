# 오늘도 올리브영! 올영세일 쇼핑몰형 재고 알림 데모

## 업로드 방법
1. `olive_young_rkdms_index.html` 파일명을 `index.html`로 변경합니다.
2. GitHub Pages repository의 최상단(root)에 업로드합니다.
3. Pages 설정이 `main / root`인지 확인합니다.
4. 배포 완료 후 GitHub Pages URL을 새로고침합니다.

## 포함 기능
- 공식 올리브영 느낌의 PC 헤더, GNB, 사이드 카테고리
- 사용자 제작 캐릭터 삽입
- 실제 올리브영 상품명/가격/이미지 URL 기반 상품 카드
- 카테고리/검색/태그 필터
- 0시/12시/18시 쿠폰 발급 및 장바구니 적용
- 장바구니 수량 변경, 삭제, 총액 계산
- 배송지 → 결제수단 → 주문완료 3단계 결제 시뮬레이션
- 마이페이지 주문/배송 조회 화면
- 발표자용 재고 하락/재입고/초기화 컨트롤
